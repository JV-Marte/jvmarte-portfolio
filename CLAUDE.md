# 💼 Freelance Portfolio Website (Claude Instructions)

## 🎯 Goal

Create a **modern, professional, and high-converting portfolio website** for a freelancer specializing in:

* GoHighLevel CRM setup
* Web development
* Automation systems

The website should:

* Be visually clean but eye-catching
* Feel premium and trustworthy
* Appeal to business owners (not just developers)
* Clearly communicate value (saving time, increasing revenue, automation)

---

## 🧠 Brand Identity

**Name:** JV (or "Your Name")
**Role:** Freelance CRM & Automation Specialist / Web Developer

**Positioning Statement:**
I build systems that help businesses capture leads, automate workflows, and grow faster.

**Core Message:**
"I turn manual processes into automated systems that save time and increase revenue."

---

## 🎨 Design Style

* Dark modern UI (black / deep blue base)
* Accent color: neon blue / electric blue
* Futuristic/tech-inspired background (subtle, not distracting)
* Smooth animations (hover, fade, scroll)
* Clean spacing and minimal clutter
